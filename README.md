# links test
## This is a repository to test if links have the same effect between operating systems

##results:

### Linux:

- [ ] Soft links propagate on OS
- [ ] Hard links propagate on OS
 
### Windows 10:

- [ ] Soft links propagate on OS
- [ ] Hard links propagate on OS

### MacOS & BSD (bsd coreutils)

- [ ] Soft links propagate on OS
- [ ] Hard links propagate on OS
